## Resumo do Projeto

O NutriTrack é um sistema web desenvolvido com o objetivo de auxiliar usuários no acompanhamento da alimentação diária por meio do registro de alimentos e controle de calorias consumidas. A aplicação foi projetada para oferecer uma experiência simples, intuitiva e acessível, permitindo maior conscientização sobre hábitos alimentares e incentivando práticas mais saudáveis.

Durante o desenvolvimento foram aplicados conceitos de Engenharia de Software, Desenvolvimento Web, Design de Interfaces, Controle de Versão com Git e gerenciamento de tarefas utilizando GitHub Projects (Kanban).

A solução final atende aos requisitos definidos na fase de planejamento e disponibiliza uma interface responsiva para facilitar o uso em diferentes dispositivos.
