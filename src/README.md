## Instruções de Utilização

### Instalação do Site

O sistema NutriTrack é um projeto web estático desenvolvido com HTML, CSS e JavaScript.  
Por esse motivo, não requer instalação complexa ou servidor backend.

Para executar o projeto, basta:

1. Baixar ou clonar o repositório do GitHub
2. Abrir a pasta do projeto
3. Executar o arquivo `index.html` em um navegador web

O sistema pode ser utilizado diretamente em qualquer navegador moderno.
